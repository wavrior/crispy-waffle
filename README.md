# alexkiele.com — portfolio

## Publish on GitHub Pages

1. Upload all files to a new repo (keep the folder structure — `img/` must stay alongside the `.html` files)
2. **Settings → Pages → Deploy from branch → main / (root) → Save**
3. Live at `https://<username>.github.io/<repo>/` within a minute
4. Custom domain: add `alexkiele.com` in Pages settings, update your DNS A record to GitHub's IPs, tick **Enforce HTTPS**
