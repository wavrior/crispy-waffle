Overview

Localising a series is straightforward when you have the project files. We didn't have any.

To deliver the French version of Human vs Hamster, I rebuilt the show's entire graphics package — 2D and 3D — from scratch, using only the English broadcast masters as reference. No project files, no original assets, no typefaces supplied. Every element was reconstructed by eye and structured as a reusable template system so that localising each new episode became a versioning job, not a rebuild.

The rebuild in frames

The test of a from-scratch reconstruction is simple: cut it against the original and see if anyone can tell. Every graphic below was rebuilt with no project files — matching typography, colour, lighting and timing from broadcast reference alone.

The build

With no project files, every asset started as detective work — pausing frames, matching typefaces, reverse-engineering motion timing and easing. The 3D name boards, logo bumps and title sequences were rebuilt from flat renders, then wrapped in a consistent template system using After Effects Essential Properties, so that an editor — or the studio's batch automation — could populate each episode's translations without touching the underlying rigs.

Even the show logo was rebuilt from scratch. Working from a broadcast screengrab, I reconstructed the wordmark as editable vector in Illustrator — every letterform, bevel and colour band — then localised it to the French Humain vs Hamster. That file became the master logo used across every graphic in the package.
